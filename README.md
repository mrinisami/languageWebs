# LanguageWebs

LanguageWebs is a Web Application which connects translators and potential clients.

## Requirements
---
- Java 17
- npm
- Docker

## Run
---
`bin up` to boot the server

## Client
---
Available commands:
`cd ui`

`npm install`
`npm start`

## Server
---
`cd src`

`./gradlew bootRun`
